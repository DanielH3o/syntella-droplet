# Syntella Marketing Site

This is a standalone public-facing landing site for Syntella.

It is intentionally separate from the product frontend and build/bootstrap flow.

Files:

- `index.html`
- `styles.css`
- `app.js`

Preview locally with any static file server, for example:

```bash
cd /Users/daniel/.openclaw/workspace/syntella/website
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.
